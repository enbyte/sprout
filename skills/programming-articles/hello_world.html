HTML Hello World

    How are webpages such as the one you are reading right now made?
    The answer is simple:
        They are written using a special language the computer can understand called HTML.
    HTML stands for HyperText Markup Language.

    Everything in HTML is made out of tags.
    Tags are used to make things on the page.
    Some tags show text, while some show big headings.
    Some tags show images, while some show videos.

    All tags look like this:
    A <, then the name of the tag, then a >, then the content, then a <, a /, the name of the tag, and >.

    For example, the p tag, used for paragraphs, looks like this:

    <p>
        This is a paragraph.
    </p>
