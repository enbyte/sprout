# Sprout

_For HawkHacks 2022._

Our project, Sprout, is an offline web app that is designed to teach STEM topics to children without access to the internet. Currently, Sprout has articles and quizzes on mathematics and science, as well as HTML, CSS, and JavaScript programming tutorials with a live web sandbox. No setup or download is necessary, and the app is entirely free.
